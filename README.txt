Cara Menjalankan Bot Telegram:

1. Install Library yang Dibutuhkan:
   pip install python-telegram-bot openpyxl

2. Buka file bot.py dan ganti:
   - ISI_TOKEN_KAMU dengan token dari @BotFather
   - ADMIN_ID dengan ID Telegram kamu

3. Jalankan bot:
   python bot.py

Catatan:
- Bot akan menyimpan semua data ke file data_user.xlsx
- Admin akan menerima notifikasi otomatis setiap kali user mengisi data
